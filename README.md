# Học viện CRT – Hệ thống Quản lý Sinh viên

Ứng dụng ôn tập tương tác được xây dựng từ báo cáo và source dự án QLSV bảo mật điểm bằng CRT.

## Xuất bản bằng GitHub Pages

1. Tải toàn bộ nội dung thư mục này lên nhánh `main`.
2. Mở **Settings → Pages**.
3. Chọn **Deploy from a branch**.
4. Chọn nhánh `main`, thư mục `/(root)`, rồi bấm **Save**.
