## Xuất bản bằng GitHub Pages

1. Tải toàn bộ nội dung thư mục này lên nhánh `main`.
2. Mở **Settings → Pages**.
3. Chọn **Deploy from a branch**.
4. Chọn nhánh `main`, thư mục `/(root)`, rồi bấm **Save**.
